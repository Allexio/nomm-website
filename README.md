# NOMM Website repo

This is the repo for the website of the [NOMM app](https://github.com/allexio/nomm).

This project was build with [Hugo](https://gohugo.io/) & [Docsy](https://github.com/google/docsy) and is hosted on [Github](https://github.com).

## Running the website locally

Building and running the site locally requires a recent `extended` version of [Hugo](https://gohugo.io). You can find out more about how to install Hugo for your environment in our [Getting started](https://www.docsy.dev/docs/getting-started/#prerequisites-and-installation) guide.

Once you've made your working copy of the site repo, from the repo root folder, run:

```bash
hugo server
```
